This German README is a Community contribution placeholder. The project name is Ilios in German for now. The Chinese name is 衔牍, the English name is Ilios, and the Japanese name is 理紐.

The actively maintained human documentation is currently [README.md](README.md) in Simplified Chinese and [README.en.md](README.en.md) in English. The Japanese README is experimental. German UI hooks already exist in the app, but this long-form README should be completed by community contributors who can keep it accurate as the product changes.

If you want to contribute German documentation, please translate from the Chinese main README or the English README, keep the local-first product boundary clear, and do not promise unsupported features such as PDF parsing, OCR, accounts, built-in sync, Docker, Redis, or Celery.
